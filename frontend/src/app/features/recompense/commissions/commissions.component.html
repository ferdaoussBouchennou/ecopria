Comment sont calculées les commissions ?

Pour les offres avec réduction (%) : valeur remise × % commission. Pour les offres de type RÉDUCTION : valeurDh × taux commission. Les offres entièrement gratuites (sans valeur monétaire) ne génèrent pas de commission.
